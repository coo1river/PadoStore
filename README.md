## <span id="intro">1. 프로젝트 소개</span>

- 세상의 모든 팬들을 위한 **거래, 공동구매 플랫폼**입니다.
- 유저들끼리 서로 판매, 구매, 교환이 가능하며 공동구매 폼을 업로드 할 수 있습니다.
- 판매, 구매, 교환 시 **채팅**을 통해 거래할 수 있습니다.
- 공동구매 폼 제출 시 유저의 **입금 폼**을 작성하며, **공동구매 관리** 또한 가능합니다.
